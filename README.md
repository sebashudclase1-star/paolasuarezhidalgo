# Paola Suárez Hidalgo

Sitio personal y portafolio de Paola Suárez Hidalgo — Community Manager y fotógrafa.

Incluye:
- `index.html`: página principal con secciones de Proyectos, Fotografía, Empresas y Contacto.

Instrucciones rápidas:

1. Revisa el contenido en `index.html`.
2. Para crear el repositorio remoto y pushear desde la línea de comandos (requiere `gh` CLI autenticado):

```bash
# desde la raíz del proyecto
gh repo create sebashudclase1-star/PaolaSuarez --public --source=. --remote=origin --push --confirm
```

Si no usas `gh`, crea el repo en GitHub y luego:

```bash
git remote add origin https://github.com/sebashudclase1-star/PaolaSuarez.git
git branch -M main
git push -u origin main
```
